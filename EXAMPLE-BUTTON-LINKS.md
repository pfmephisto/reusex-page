# Example Button Link Configurations

Here are some real-world examples of button configurations you might want to use:

## E-commerce / SaaS Platform

```yaml
hero:
  cta_primary:
    text: "Start Free Trial"
    url: "https://app.yourcompany.com/signup"
  cta_secondary:
    text: "View Pricing"
    url: "/pricing"

cta:
  primary_button:
    text: "Get Started Now"
    url: "https://app.yourcompany.com/register"
  secondary_button:
    text: "Talk to Sales"
    url: "https://calendly.com/yourcompany/sales-demo"
```

## Service Business

```yaml
hero:
```


## Common URL Types

- **App/Platform**: `https://app.yourcompany.com/signup`
- **Calendly**: `https://calendly.com/yourcompany/meeting-type`
- **Google Forms**: `https://forms.gle/your-form-id`
- **Typeform**: `https://yourcompany.typeform.com/to/form-id`
- **Email**: `mailto:contact@yourcompany.com`
- **Phone**: `tel:+1-555-123-4567`
- **Internal Pages**: `/pricing`, `/about`, `/contact`
- **Page Sections**: `#about`, `#features`, `#demo`
